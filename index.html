世代別ノート（デモ）

あなたの年代（高校生・大学生・社会人など）を選ぶと、
よくある悩み・心理学のキーワード・「二周目目線の捉え直し」が表示されます。

  • 例：大学生 → 就活不安 / 社会的比較 / アイデンティティの揺れ
  • 例：社会人 → バーンアウト / キャリアの曲がり角 / 同期との比較

※現在はサンプル表示です。本実装では年代ごとの詳細データを紐づけ予定。

教養ノート（デモ）

歴史・科学・心理学・社会の出来事を、
「今の自分の人生にどう関係するか？」という視点でまとめていくノートです。

  • 産業革命 → 「識字率」と現代の「AIリテラシー」を対応させて理解
  • 行動経済学 → 自分の買い物・勉強・人間関係のクセを言語化
  • 社会構造の変化 → AI時代のキャリア設計に落とし込む

※今は固定テキストですが、将来はトピックごとにカード化予定。

キャッチアップノート（デモ）

1日5分で、世界と日本・AI・社会変化の「今だけ大事なところ」だけ拾うページ。
気になったトピックは教養ノートや世代別ノートに送って深掘りできます。

  • 今日のニュース3本（AI要約）
  • 今日のキーワード：例）「生成AIエージェント」「リスキリング」
  • 自分メモ：「これは将来の自分に関係しそう / しなさそう」

※今はサンプル文章です。API連携で自動更新を予定。
